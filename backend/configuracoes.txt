# 2ManyTareas - Backend

### Variáveis de ambiente
- `DATABASE_URL` = `mysql+pymysql://usuario:senha@host:3306/2manytareas`
- `SECRET_KEY` = uma string aleatória
- `CORS_ORIGINS` = `*` (ou domínio do front)

### Rodar local
```bash
python -m venv .venv
. .venv/Scripts/activate   # Windows (ou source .venv/bin/activate no Linux/Mac)
pip install -r requirements.txt

# crie o banco 2manytareas no MySQL localmente
# atualize usuario/senha/host na config (ou defina DATABASE_URL)

python app.py
